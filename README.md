## 📖 Project Description

The AI Stock Forecast App is a machine learning-powered financial analytics dashboard developed using Streamlit.

The application fetches real-time stock market data, analyzes historical stock performance, and predicts future stock prices using Machine Learning techniques.

The dashboard provides interactive visualizations, moving average analysis, trend detection, market insights, and AI-based forecasting in a modern and user-friendly interface.

---

## 🎯 Objective

* Analyze historical stock market data
* Predict future stock prices using Machine Learning
* Visualize market trends interactively
* Assist users in financial analysis and decision-making
* Demonstrate practical AI applications in finance

---

## 🧠 Machine Learning Model

* Algorithm: Linear Regression
* Features Used:

  * Close Price
  * Lag 1 Price
  * Lag 2 Price
* Data Source: Yahoo Finance API
* Libraries: Scikit-learn, Pandas, NumPy
